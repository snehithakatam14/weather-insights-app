# Weather Forecasting Data Analysis Application

##  Problem Statement
Weather data is essential for planning and decision-making. This project focuses on retrieving real-time weather data for a given city and analyzing key weather parameters to provide meaningful insights through visualizations.

##  Data Source
- OpenWeather API (real-time weather data)

##  Tools & Technologies
- Python
- Streamlit
- Matplotlib
- OpenWeather API

##  Project Workflow
1. Fetches real-time weather data using API requests
2. Extracts important weather features such as temperature and humidity
3. Performs basic data processing
4. Visualizes weather trends using plots
5. Displays insights through an interactive interface

##  How to Run the Project
1. Clone the repository  
2. Install dependencies using `pip install -r requirements.txt`  
3. Run the application using `streamlit run app.py`

##  Conclusion
This project demonstrates the use of Python for data collection, processing, and visualization using real-world API data.
